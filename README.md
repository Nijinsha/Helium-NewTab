# Helium New Tab

A custom new tab extension for Helium browser featuring a search bar and gradient background.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" (toggle in the top right)
3. Click "Load unpacked"
4. Select this directory
5. Open a new tab to see the custom page

## Features

- Custom gradient background
- Search bar that redirects to Google search
- URL detection - enter a URL to navigate directly
- Clean, minimal design
